# Usage
First Time - Requires Python 3.2+  
```sh
git clone https://github.com/TorchedSammy/YTLink
cd YTLink
pip install youtube-search
python main.py -h
```  

With [youtube-dl](https://github.com/ytdl-org/youtube-dl):  
```sh
youtube-dl $(python main.py <QUERY>)
```
